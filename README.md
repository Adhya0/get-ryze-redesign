# Get-Ryze.ai Website Redesign

This project is a redesigned version of the Get-Ryze.ai website, with a focus on clarity, user experience, and modern SaaS design.

## Design & Structure
- Clean, professional layout with green accents
- Modular React components for scalability
- Separate pages for Features, Pricing, Contact
- Clear call-to-action buttons (Book a Demo,Get started)

## Tech Stack
- React + Vite
- React Router
- CSS / Bootstrap
